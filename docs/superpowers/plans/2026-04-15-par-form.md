# `(par ...)` Parallel Step Execution — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add `(par ...)` as a compound form that runs child steps concurrently via goroutines with fail-fast semantics.

**Architecture:** New `par` compound form in the sexpr parser produces a `Step` with `Form: "par"` and `ParSteps []Step`. The runner forks goroutines via `errgroup.WithContext`, guards `rctx.steps` with a mutex, and collects telemetry per-goroutine. Works anywhere a step goes: top-level, inside phases, inside map bodies.

**Tech Stack:** Go, `golang.org/x/sync/errgroup`, `sync.Mutex`

**Spec:** `docs/superpowers/specs/2026-04-15-par-form-design.md`

```go
// convertPar: (par (step ...) (step ...) ...)
// All children run concurrently. Minimum 2 children.
func convertPar(n *sexpr.Node, defs map[string]string) (Step, error) {
	children := n.Children[1:] // skip "par"
	if len(children) < 2 {
		return Step{}, fmt.Errorf("line %d: (par) needs at least 2 children", n.Line)
	}

	var parSteps []Step
	for _, child := range children {
		if !child.IsList() || len(child.Children) == 0 {
			return Step{}, fmt.Errorf("line %d: (par) children must be forms", child.Line)
		}
		head := child.Children[0].SymbolVal()
		if head == "" {
			head = child.Children[0].StringVal()
		}
		steps, err := convertForm(child, head, defs)
		if err != nil {
			return Step{}, fmt.Errorf("line %d: par child: %w", child.Line, err)
		}
		parSteps = append(parSteps, steps...)
	}

	return Step{
		ID:       fmt.Sprintf("par-%d", n.Line),
		Form:     "par",
		ParSteps: parSteps,
	}, nil
}
```

```go
// executePar runs all child steps concurrently and waits for completion.
// Fail-fast: first error cancels all siblings via context.
func executePar(ctx context.Context, rctx *runCtx, step Step) (*stepOutcome, error) {
	g, gctx := errgroup.WithContext(ctx)

	type parResult struct {
		id      string
		outcome *stepOutcome
	}
	results := make([]parResult, len(step.ParSteps))

	for i, child := range step.ParSteps {
		i, child := i, child
		g.Go(func() error {
			outcome, err := executeStep(gctx, rctx, child)
			if err != nil {
				return fmt.Errorf("par step %q: %w", child.ID, err)
			}
			results[i] = parResult{id: child.ID, outcome: outcome}
			return nil
		})
	}

	if err := g.Wait(); err != nil {
		return nil, err
	}

	// Store all outputs (executeStep already stores via mutex, but par wrapper
	// ID also needs an output — concatenate child outputs)
	var combined []string
	for _, r := range results {
		if r.outcome != nil {
			combined = append(combined, r.outcome.output)
		}
	}

	out := strings.Join(combined, "\n")
	rctx.mu.Lock()
	rctx.steps[step.ID] = out
	rctx.mu.Unlock()

	return &stepOutcome{output: out}, nil
}
```
